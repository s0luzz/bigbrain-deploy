## contrast and readability
- text always has good contrast like white on dark or gray on light
- correct and wrong answers use strong green or red colors

## keyboard navigation support
- all buttons are real html buttons so you can tab and press enter

## focus states and indicators
- all buttons have visible outlines and hover states
- buttons are big enough for mobile or keyboard users

## semantic html usage
- uses tags like nav button input ul and video
- image tags have alt text like the game name on dashboard

## reduced motion and visual clarity
- there are no animations or flashing elements
- layout uses clean spacing so its easy to read
